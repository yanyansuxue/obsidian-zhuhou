## 童年
```timeline
[line-3,body-5,active-color-text-accent]
+五岁
+ 野心伊始
+ 商长嬴生在封邑，那是一处苦寒之地，她的父亲燕胥并不受国君宠爱，因而封地荒僻，但比这更荒凉的是人心。人生的前五年，她都在绞尽脑汁地活着，直到五岁那年的冬天，她实在太冷，躲到燕胥住处的窗沿下，希冀于能蹭到屋里一点暖气，却意外听到他与幕僚之间的议事，她旁听了几日，敲响了燕胥住所的房门，在二人诧异的目光下反驳了幕僚所献之策。出乎意料的，燕胥斥责了她，但又让她日后进屋读书。她不知道什么是读书，但能进屋意味着不用受冻，于是应下了。

+ 八岁
+ 赴姜为质
+ 商长嬴跟随燕胥学了三年，知道了虞国国都在丰都，姜王朝的都城在未阳，八岁那年她听人说丰都有个堂兄被选去未阳当质子，可堂兄不愿，为此生出许多争执，于是她主动到燕胥跟前请缨，自陈她愿意替兄赴姜。说来也巧，那个堂兄燕同与她同年同月出生，然而二人的命运天差地别——国君认为燕寅之男燕同是先太子燕穆的转世，对他殊为宠爱，有求必应，也正因此国君舍不得让心爱的孙儿远赴姜都，听到商长嬴愿去，又见她身量与燕同所差不大，便轻而易举地应下了此事，命她女扮男装，赴姜为质。
  
  +八岁-十一岁
  +质子三年
  +野心就如野草越烧越旺，商长嬴先是被虞宫的豪奢所惊，转而又发现所谓虞宫在姜王宫面前不值一提。质宫中汇集着诸国质子，商长嬴拼命地汲取知识，但因弱国质子的身份屡遭冷眼。直到几个诸侯国的一场叛乱，姜王朝此时余晖尚在，姜天子镇压了叛乱，并命诸侯国质子自尽，就在这时，商长嬴站了出来，主动砍下了几名质子的头颅。
死寂过后姜天子大笑，问她名姓，又将她指到姜太子晋雍身边做伴读。

+十一岁
+相马
+十一岁燕胥筹谋宫变，商长嬴借口归国，并在宫变中出力。此次宫变国君被诛、燕胥之弟公子寅及其妻子被囚，燕胥名正言顺登位。
不久后商长嬴从燕寅口中得知真相，她实则并非燕胥的亲生女儿，多年前先太子燕穆病逝，燕寅之妻庄夫人不久后怀孕，坊间传言此子为先太子转世，国君沉浸在丧子之痛中，对这个说法深信不疑，为了圆这个谎言，燕寅将刚出世的商长嬴与燕胥之男相换，有了如今深受宠爱的燕同。商长嬴痛恨之下亲口命人杀掉燕寅及其妻子，自己也跪倒在雪地里干呕不止。
此时燕同对外仍为被囚之身，姜天子特地遣使命燕同归姜，太子晋雍也自请随行。在十一岁这年，商长嬴面临着三个选择——燕寅诱之以血缘，燕胥诱之以利益，晋雍诱之以情谊，惟有商长嬴本人对此感到愤怒，她愤怒于任何人试图掌控她，也愤怒于这三个人同时如此轻视她，所以她假意答应晋雍，背后设局在途经宋国时假死，彻底杀死了燕同这个马甲。不知真相的晋雍大恸，归姜后联合虞国攻打宋国，虞国因此获地。在史书之外，商长嬴隐姓埋名，饮血嚼肉，捱过又一个极寒的凛冬后，在春日里朝着丰都策马驰驱。
```
## 青年

```timeline
[line-3, body-5,active-color-text-accent]
+十四岁
+ 赴荆为质
+燕胥登位后，着手治理虞国，然而虞国地僻，邻国荆国意识到虞国国力日强，感到危机而与之交战。此战虞国败附于荆，为彰其诚信，燕胥嫡长子燕冉赴荆为质，向夫人苦求无果，只能目送长子去国。
这年商长嬴与商荷华、燕冉一道赴荆，名为质实为媵，两国议定将商荷华许给崔伯，荆国由大夫栾非执政多年，栾非之子栾朋见商荷华美而自娶，商长嬴与商荷华私下筹谋，使商荷华引诱栾朋之弟栾成与之相争，太傅陈忌趁栾家内讧时夺权，栾非被杀，燕胥于此时突然兴兵发难，陈忌大怒，方知燕胥佯败，欲烹杀燕冉，却得知商长嬴、商荷华姊妹早已带着燕冉逃离荆都。

+十五岁
+子樛
+虞国再次兴兵伐荆，荆国无力招架，被迫割土议和，商长嬴姐弟三人安然归虞，燕胥设宴庆贺，适逢商长嬴及笄，笄礼上燕胥赠之金笄，为之字“子樛”。子樛者，取“南有樛木，葛藟萦之”，既寓虞国地北，对南下虎视眈眈，也寓“乐只君子，福履绥之”，取美好祝福之意。是日众人齐聚，夜晚围坐火前，商长嬴为之剑、商荷华为之乐、燕冉为之歌，燕胥及向夫人等女眷共赏，尚年幼的商婵、燕偃模仿兄姊歌舞。那是难得的一夜，众人透过命运的罅隙短暂忘却名姓，抛却恩怨利益，只因同为家人享一晌和乐，而后又各自如东逝水，裹挟着注定的命运滚滚向前。
虞国国土扩张，逐渐为中原诸侯国所侧目。姜天子晋旷勇而无谋，鲁国新王即位后不来朝见天子，晋旷大怒，命人烹杀质宫中的鲁质子，并将肉送到鲁国请鲁王品尝，此举无疑违背姜礼，鲁王联合盟国声讨姜天子，双方就此交战。晋旷以质宫质子要挟其余诸侯国参战，然而人心惶惶，屡战屡败，晋旷亲征而摔下马，致使跛足卧床，愈发喜怒无常，鲁王取姜数百里疆域而去，自称霸王，从此不再朝姜，乱世由此开始。
卫国是距离虞国最近的大诸侯国，燕胥有意与之结交，商长嬴主动请缨出使。卫国国君桓祯尚且年幼，朝中由其叔公子嘉主政，她听闻公子嘉善弈，于是在雪中来访，与之对弈，二人对论经史、畅谈大势，聊到相见恨晚，桓嘉问她此行有何所求，商长嬴说自己久居苦寒之地，想等秋收时在田垄上看一回风吹稻浪，桓嘉欣然应允，将她留在宫中，以礼待之。时日久了，坊间流言四起。商长嬴在卫国作风跋扈，数名臣子因得罪她而获罪下狱，桓嘉亦不闻不问。是年秋日，桓嘉与燕胥在葛城结盟，公女荷华联姻卫国，以彰两国之好。

+十六岁
+舞邑监兵
+次年商长嬴归虞，适逢戎族侵犯边地，商长嬴向燕胥举荐近侍陈午，燕胥允，并命商长嬴前往舞邑监兵。历来公子极少亲自带兵，这被认为是遭到国君厌弃的征兆，商长嬴听后面无愠怒，呈上一卷缣帛，对燕胥说这是她在卫国请教后所撰写的农书，虞国地广人稀，农牧并重，土地大量荒废，若能大兴耕种，则兴国可待。燕胥动容，对商长嬴承诺，若是此战大捷，寡人定封你为卿。商长嬴离宫时正好遇到商婵，商婵问她要去哪里，商长嬴说自己要去舞邑，商婵年纪尚幼，听不懂言下之意，闹着也想去，商长嬴就把她带到马车上，到了下一座城告诉她这是舞邑，等商婵玩累睡着了就命人将她送回去。

+十六岁-二十二岁
+舞商
+军中将士起初对商长嬴多有异议，但商长嬴与士卒同吃同住，对待最低级的小卒也谦逊有礼，被人挑衅也毫无怒色，逐步赢得士卒的信任。当时军队如何行进要听从国君和正卿的指令，商长嬴遇事不能擅自发令，必须请示国君的命令，然而戎族来势汹汹，虞军兵力不及节节败退，陈午向商长嬴说应当退至山中，商长嬴向燕胥请示，朝中多有反对，燕胥迟迟未下令，商长嬴便采纳陈午的意见，未经允许便让士卒退守山中，戎族贸然冲入山林，遭到伏击，虞军重创戎族，乘胜追击，大捷而归。这时商长嬴向燕胥请罪，自述不孝，朝中亦多有指责声，认为按照律法她应该受刑甚至斩首，唯独陈午上书请求宽宥，卫国公子嘉听闻此事后，亦派人来给商长嬴求情，燕胥就命她守在舞邑将功折罪。
商长嬴于是在舞邑住下，大兴农业，普及教化，严明军纪，在她的治理下，舞邑欣欣向荣，几年都没有被戎族所扰，赢得当地民众的爱戴，尊称她为“舞商”。而在商长嬴被远封舞邑的光阴里，虞国同样韬光养晦。次年，晋旷驾崩，晋雍即位。同年，桓祯成年，桓嘉还政，离开卫国四处悠游。

+二十二岁-二十三岁
+攻卫
+商长嬴二十二岁那年，卫国对荆国发动战争，如今小诸侯国屡遭吞并，荆国成为虞、卫两国之间唯一的诸侯国，一旦荆国国破，虞国将与卫国正面对上。商长嬴听后请求回到丰都，自称能够击退卫国，燕胥召回她问策，阔别六年，商长嬴一反昔年冷淡清瘦的模样，身形高大，英武逼人，在她的带领下，卫国军队成功被逼退。而商长嬴又在此时上书，说如今虞国粮食充盈，足以支持长久作战，请求乘胜追击，与卫国正式开战。
虞卫交战一年，桓祯威望不足，识人不清，卫国节节败退，桓嘉姗姗来迟，尽管亲往前线仍无力回天。次年，卫国国都被破，桓祯被杀，桓嘉流亡蔡国，商长嬴携商荷华归国。交战前期，虞国形势严峻，燕冉以身诱敌，为虞国迎来喘息之机，不幸战死。
经此一役，虞国一跃成为大国。商长嬴居功至伟，受封上将军，重回宫中。

+二十三岁
+离心
+不久后燕胥召见她慰问，期间忽然意外昏倒，商长嬴急召人入内诊断，太医禀明是操劳所致，商长嬴命人奉上汤药亲自喂下，期间燕偃闻讯赶来，想要入内侍疾，被商长嬴以需要静养的理由拒绝。燕胥醒来后询问情况，商长嬴如实回答，燕胥听后大怒，斥其不孝不悌，商长嬴惶恐问罪，泫然流涕，面无愠色，燕胥方神情稍缓。如是数次，商长嬴皆无怨言，左右却渐生流言，多对商长嬴敬而远之。

+二十四岁
+宫变登位
+次年，鲁王遣使访虞，想要联合虞国再次攻打姜都。燕胥没有说是否同意，鲁王听后很不快，于是联合盟国攻姜，结果晋雍听从桓嘉的建议，成功逼退鲁国。同年，桓嘉被晋雍封到蔡国，桓嘉带兵杀死蔡侯，自己取而代之。
是年，听闻燕胥风寒，商长嬴入宫觐见。行至榻前，忽然伸手扼住燕胥的喉咙，燕胥在此时猛然睁眼，挥刀直刺，商长嬴被迫松手，险险躲开。与此同时，商长嬴的精兵杀入武库，夺取武装，与禁卫厮杀在一起；陈午率兵控制宫门，封锁消息；商荷华前往内廷，扣下官员的印章。商长嬴虽占得先机，但燕胥手持匕首，一时二人缠斗，不分伯仲，宫人不敢上前，直到燕胥用匕首刺穿商长嬴的左臂，商长嬴后退被一名内侍擒住，让燕胥寻到机会，将要挥刀刺下时，却先被另一个匕首贯穿胸口，在他身后，向夫人手持沾血的匕首，脸色煞白。是日，商长嬴控制宫廷，同时血洗南家满门，公子偃被逐出虞宫。
商长嬴登位，谥父为庄，是为虞庄王。

+二十四岁-二十五岁
+ 君相协契
+商长嬴登基前便患有心理疾病，在登基前一年与燕胥对峙时更为严重，这种心理疾病来源于童年阴影（十一岁弑杀生身父母）+战争PTSD（弟弟战死）+燕胥精神打压与高压监视+自身性格缺陷（常年高压环境导致防备心重，很难与人交心，压力无法纾解），在她登基后任命商荷华为相，并由商荷华代理政务。商荷华雷厉风行，刚柔并济，兼之商婵辅助，迅速使混乱政局步入正轨。次年，商婵自请和亲鲁国。

```
## 中老年
```timeline
[line-3, body-5,active-color-text-accent]
+二十五岁-四十六岁
+南北称帝
+商长嬴病情好转，重新着手处理政务，数年后南下攻姜，率北方诸侯逐姜天子于淇水之外，一年后结盟议和，与姜天子相约南北称帝，划江而治，时人以北帝相称。公子偃作为质子被派到南姜。商长嬴统治北方，勤政明法，力行变法，收拢诸侯，逐渐集权，北方从分封制向封建制转变，史称圣武改制。商长嬴三十三岁，商玖出生。数年后公子偃归北姜，负责教导商玖。

+四十七岁
+梧桐半死
+商荷华病逝，帝怆恸，以天子礼葬之，追封舜文后。从此以后，世间梅花无并蒂，青史长留凤凰声。

+五十七岁
+嬗位离宫
+商荷华死后十年，商长嬴结束三十三年的帝王生涯，嬗位商玖，离开虞宫，仅有数名无官位、无声名的老人自愿追随，此后开始流浪，再无音讯。商玖即位之初，一度大肆清洗商长嬴势力。

+六十七岁-六十八岁
+生命尽头
+十年后，商长嬴曾秘密回京，私下与商婵、商玖小聚，没有惊动他人，那是她最后现于人世。不到半年后，追随者回宫觐见，告知商长嬴已在睡梦中安然离世，享年六十八岁，并将遗体带回宫中。
翌日，商婵自缢。商玖将商长嬴、商婵与商荷华共葬株林山，谥帝曰圣武，树碑铭文，永祀太庙。

+死后
+身后事
+商长嬴死前要求简葬，唯独带着部分旧物及大量文章入土，其中有当时的律法文章，有她登基前的种种筹谋，有登基后与各路诸侯的互通有无，与臣属部下的交流，也有嬗位后的大量闲笔，记叙了这十一年来的所见所闻，以及她对生命的思考与怀念，这些文章都由追随者汇集起来，随着她的入土而隐秘于世。史书记载，她与商荷华、商婵合葬于株林山，但商玖的选址极其刁钻，历代盗贼想要盗墓都无功而返，直到现代运用仪器才探测到墓葬位置。

墓中还有一把旧剑，剑名止戈。

商长嬴死后数年，商玖率兵南下，山河易名。

商长嬴死后千年，王朝更替，天下数变，门户私计与天下大同此消彼长，株林山上水草丰茂，牛羊成群。

```
